1. 两数之和

给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。

你可以假设每种输入只会对应一个答案。但是，你不能重复利用这个数组中同样的元素。

思路：空间换时间，用hashmap存储他们的值和下标，遍历一遍

  public int[] twoSum(int[] nums, int target) {
        HashMap<Integer,Integer> map=new HashMap<>();

        for (int i = 0; i < nums.length; i++) {
            int complement=target-nums[i];
            if(map.containsKey(complement)){
                return new int[]{map.get(complement),i};

            }
            map.put(nums[i], i);
        }
        return new int[0];

    }

4.最长的回文子串

题目；给定一个字符串 s，找到 s 中最长的回文子串。你可以假设 s 的最大长度为 1000。
示例：

输入: "babad"
输出: "bab"
注意: "aba" 也是一个有效答案。

思路:动态规划，用一个二维boolean数组A[i][j]记录从下标i到j是否是回文串，如果A[i][j]是回文串那么A[i+1][J-1]也是回文串。先单独计算相邻两个字符是否是相等的，也就是计算长度为2的回文串，然后以长度3的滑动窗口去扫描字符串，判断是否是回文串，依次增大滑动窗口的大小，直到=length；

public String longestPalindrome(String s) {
        if(s==null||s.length()==0){
            return "";
        }
        int length=s.length();
        boolean[][] isPalindrome=new boolean[length][length];
        char[] chars=s.toCharArray();
        int palindromeStart=0;
        int palindromeEnd=0;
        for (int i = 0; i <length ; i++) {
            isPalindrome[i][i]=true;
            for (int j = i; j <length-1 ; j++) {
                if(chars[j]==chars[j+1]){
                    isPalindrome[j][j+1]=true;
                    palindromeStart=j;
                    palindromeEnd=j+1;
                }

            }
        }

        for (int l = 3; l <=length ; l++) {//长度为length结束
            for (int start = 0; start+l-1 <length ; start++) {
                    int end=start+l-1;
                    if(chars[start]==chars[end]&&isPalindrome[start+1][end-1]==true){
                        isPalindrome[start][end]=true;
                        palindromeStart=start;
                        palindromeEnd=end;
                    }
            }
        }
        return s.substring(palindromeStart,palindromeEnd+1);
    }

为什么不一起计算：因为当长度为2，即j=i+1，isPalindrome[start+1][end-1]无效，会越过彼此。

15. 三数之和

给定一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？找出所有满足条件且不重复的三元组。

注意：答案中不可以包含重复的三元组

例如, 给定数组 nums = [-1, 0, 1, 2, -1, -4]，

满足要求的三元组集合为：
[
  [-1, 0, 1],
  [-1, -1, 2]
]

思路：重点在与不重复，如何去重，将数组排序，然后从头开始遍历，定义两个指针，指向当前下标的前一个位置和最后面的位置，

public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>>  lists=new ArrayList<>();
        if (nums == null||nums.length==0) {
            return lists;
        }
        Arrays.sort(nums);
        int length=nums.length;
        if(nums[0]<=0&&nums[length-1]>=0) {//为了包含数组全是0的情况
            for (int i = 0; i < length ; i++) {
                if(i>0&&nums[i]==nums[i-1]){//从第二位开始去重
                    continue;
                }

                int start = i+1;
                int end = length - 1;
                while (start < end) {
                    int result = nums[start] + nums[end] + nums[i];
                    if (result == 0) {
                        List<Integer> list = new ArrayList<>();
                        list.add(nums[start]);
                        list.add(nums[end]);
                        list.add(nums[i]);
                        lists.add(list);
                        while (start < end && nums[start] == nums[start + 1]) {
                            start++;
                        }
                        while (end > start && nums[end] == nums[end - 1]) {
                            end--;
                        }
                        start++;
                        end--;
                    } else if (start < end && result < 0) {
                        start++;
                    } else if (end > start && result > 0) {
                        end--;
                    }
                }
            }
        }

        return lists;
    }

public int myAtoi(String str) {
        str=str.trim();
        if (str == null || str.length() == 0) {
            return 0;
        }

        char[] chars=str.toCharArray();
        int length=chars.length;
        boolean isPositive=true;
        int i=0;
        if(chars[i]=='-'){
            i++;
            isPositive=false;
        }
        if(chars[0]=='+'){
            i++;
        }
        long combine= 0;
        for (; i < length; i++) {
            int digit=chars[i]-'0';
            if(digit>=0&&digit<=9){
                combine*=10;
                combine+=digit;
                if(isPositive&&combine>Integer.MAX_VALUE){
                    return Integer.MAX_VALUE;
                }
                if(!isPositive&&combine>Integer.MAX_VALUE){
                    return -Integer.MIN_VALUE;
                }
            }else {
                break;
            }

        }
        return (int) (isPositive?combine:-combine);
    }

2.两数相加

给出两个 非空 的链表用来表示两个非负的整数。其中，它们各自的位数是按照 逆序 的方式存储的，并且它们的每个节点只能存储 一位 数字。

如果，我们将这两个数相加起来，则会返回一个新的链表来表示它们的和。

您可以假设除了数字 0 之外，这两个数都不会以 0 开头。

输入：(2 -> 4 -> 3) + (5 -> 6 -> 4)
输出：7 -> 0 -> 8
原因：342 + 465 = 807

思路：注意看清题目：是逆序，也就说头结点是最小的

public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        ListNode pNode=new ListNode(0);
        ListNode pHead=pNode;
        ListNode p=l1;
        ListNode q=l2;
        int carry=0;
        while (p!=null||q!=null){
            int x=p!=null?p.val:0;
            int y=q!=null?q.val:0;
            int sum=x+y+carry;
            carry=sum/10;
            ListNode pNext=new ListNode(sum%10);
            pNode.next=pNext;
            pNode=pNode.next;
            if(p!=null)
                p=p.next;
            if(q!=null)
                q=q.next;
        }
        if(carry!=0){
            ListNode tail=new ListNode(1);
            pNode.next=tail;
        }
        return pHead.next;  
    }

23.合并k个有序链表

合并 k 个排序链表，返回合并后的排序链表。请分析和描述算法的复杂度。
思路：用最小堆，将每个链表的头结点加入进去，每次取出最小值

 public ListNode mergeKLists(ListNode[] lists) {
       if(lists==null||lists.length==0){
           return null;
       }
        PriorityQueue<ListNode> heap = new PriorityQueue<>(Comparator.comparingInt(ListNode->ListNode.val));//构造int比较器
        int length = lists.length;
        for (int i = 0; i < length; i++) {
            if(lists[i]!=null){
            heap.offer(lists[i]);
            }
        }
        ListNode pCurrent = new ListNode(0);
        ListNode pHead = pCurrent;
        while (!heap.isEmpty()) {
            ListNode pNode = heap.poll();
            pCurrent.next = pNode;
            pCurrent = pCurrent.next;
            if (pNode.next != null) {
                heap.offer(pNode.next);
            }
        }
        return pHead.next;
    }

61.旋转链表

题目：给定一个链表，旋转链表，将链表每个节点向右移动 k 个位置，其中 k 是非负数。

输入: 1->2->3->4->5->NULL, k = 2
输出: 4->5->1->2->3->NULL
解释:
向右旋转 1 步: 5->1->2->3->4->NULL
向右旋转 2 步: 4->5->1->2->3->NULL

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/rotate-list
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

思路：先将链表连成环，然后新的头结点就在n-k处，n是链表的节点个数，k指的是对n求余。

 public ListNode rotateRight(ListNode head, int k) {
        if (head == null || k < 0) {
            return null;
        }
        //convert to circle
        ListNode pHead = head;
        ListNode pNode = head;
        int countOfNode = 1;
        while (pNode.next != null) {
            countOfNode++;
            pNode = pNode.next;
        }
        pNode.next = pHead;
        int index = countOfNode - (k % countOfNode);
        for (int i = 0; i < index; i++) {
            pNode = pNode.next;
        }
        ListNode newHead = pNode.next;
        pNode.next = null;

        return newHead;
    }

11.盛最多水的容器

给定 n 个非负整数 a1，a2，…，an，每个数代表坐标中的一个点 (i, ai) 。在坐标内画 n 条垂直线，垂直线 i 的两个端点分别为 (i, ai) 和 (i, 0)。找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。
也就是求构成的最大面积
思路：暴力法复杂度为O（n），可采用双指针（l，r），首先指向数组的头尾，比较height[l]height[r]的长度，我们知道移动长的那条是不会影响矩形面积的，应该说不会增大，所以每次移动短的那根。

public int maxArea(int[] height) {
        if(height==null||height.length<2){
            return 0;
        }
        int length=height.length;
        int l=0;
        int r=length-1;
        int max=0;
        while (l<r){
            max=Math.max(max,(r-l)*Math.min(height[r],height[l]));
            if(height[r]>height[l]){
                l++;
            }else {
                r--;
            }
        }
        return max;

    }

43.字符串相乘

题目：给定两个以字符串形式表示的非负整数 num1 和 num2，返回 num1 和 num2 的乘积，它们的乘积也表示为字符串形式

思路：示例

num1="24" num2="48"
那么就等于
        32
   +   16
   +  16
   +  8 
   =  9792
当num1的下标为i，num2的小标为j，那么想乘后的数字在新的数组的位置就是i+j和i+j+1，注意i+j+1是低位，然后每次得到的乘积加上之前乘积的高位就是新的乘积，对他进行高位低位分配，因为两个一位数相乘最多只有两位
public String multiply(String num1, String num2) {
        if(num1==null||num2==null||num1.length()==0||num2.length()==0){
            return "";
        }
        if(num1.equals("0")||num2.equals("0")){
            return "0";
        }
        int length1=num1.length();
        int length2=num2.length();
        char[] str=new char[length1+length2];
        for (int i = 0; i < length1+length2; i++){
            str[i]='0';//如果不赋值为‘0’，那么它就是ASCALL里的0，为‘’空值
        }
        char[] chars1=num1.toCharArray();
        char[] chars2=num2.toCharArray();
        for (int i = length1-1; i >=0 ; i--) {//下标大的为低位
            for (int j = length2-1; j >=0 ; j--) {
                int mul=(chars1[i]-'0')*(chars2[j]-'0');
                int p1=i+j;int p2=i+j+1;int sum=mul+(str[p2]-'0');//加上低位原来的数

                str[p2]= (char) (sum%10+'0');
                str[p1]+= sum/10;//这里就不用加‘0’了
            }
        }
        int i=0;
        for (; i <length1+length2 ; i++) {
            if(str[i]!='0'){
                break;
            }
        }

        return String.valueOf(str,i,length2+length1-i);

    }

9.回文数

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。负数不是回文数

思路：官方解法：比如123321，翻转一半的数字，变成123和123，然后判读两数是否相等，如果12321，两数变成12，和123，则判断x==reverseNum/10；

 public boolean isPalindrome(int x) {
        if(x<0||(x%10==0&&x!=0)){
            return false;
        }
        int reverseNum=0;
        while (reverseNum<x){
            reverseNum=reverseNum*10+x%10;
            x/=10;
        }
        return x==reverseNum||x==reverseNum/10;
    }

147.排序链表

在 O(n log n) 时间复杂度和常数级空间复杂度下，对链表进行排序。

思路：归并排序，找出中间节点，然后划分，然后对两边的数字进行排序。
快慢指针可以找出中间节点：我的理解：不从链表的第一个节点开始，而是从链表外看，慢指针第一次跳一格，那么就落在第一个节点，快指针一次跳两格，那么就跳在第二个节点，那么进行n次，那么慢指针落在n处，快指针落在2n步

 public ListNode sortList(ListNode head) {
            if(head==null||head.next==null){
                return head;
            }
            ListNode slow=head;
            ListNode fast=head.next;
            while (fast!=null&&fast.next!=null){
                slow=slow.next;
                fast=fast.next;
                if(fast!=null){
                    fast=fast.next;
                }
            }
           ListNode right=sortList(slow.next);
            slow.next=null;//拆分成两段
            ListNode left=sortList(head);
            ListNode p=new ListNode(0);
            ListNode newHead=p;
            while (left!=null&&right!=null){
                if(left.val<right.val){
                    p.next=left;
                    left=left.next;
                }else {
                    p.next=right;
                    right=right.next;
                }
                p=p.next;
            }
            p.next=(left==null?right:left);
            return newHead.next;
    }

假设按照升序排序的数组在预先未知的某个点上进行了旋转。

思路：
往后规约有两种情况：

  1. nums[start]<nums[mid],说明前面部分是升序的，那么如果target>nums[mid]或者<nums[start]。那么就往后规约
  2. nums[start]>nums[mid],说明后面部分是升序的，那么如果target>nums[mid]并且<nums[start]。那么就往后规约
  3. 其他情况往前规约
    public int search(int[] nums, int target) {
     if(nums==null||nums.length==0){
         return -1;
     }
     int start=0;
     int length=nums.length;
     int end=nums.length;
     while (start<end){
         int mid=(start+end)/2;
         if(nums[start]<nums[mid]&&(nums[start]>target||nums[mid]<target)){
             start=mid+1;
         }else if(target>nums[mid]&&target<nums[start]){
             start=mid+1;
         }else if(nums[mid]==target){
             return mid;
         }
         else {
             end=mid-1;
         }
     }
     return start==end&&start<nums.length&&(nums[start]==target)?start:-1;
 }

二叉树的最近公共祖先

给定一个二叉树, 找到该树中两个指定节点的最近公共祖先。

百度百科中最近公共祖先的定义为：“对于有根树 T 的两个结点 p、q，最近公共祖先表示为一个结点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。”

例如，给定如下二叉树: root = [3,5,1,6,2,0,8,null,null,7,4]

思路：

  1. 找到从根节点到p，q的两条路径，然后寻找最后一个相同的节点，复杂度较高，而且需要额外空间

             return lists.get(0).get(i);
         }

 }

 public void core(TreeNode root, TreeNode p, TreeNode q) {
     if (root == null) {
         return;
     }
     list.add(root);
     if (root == p || root == q) {
         List<TreeNode> path = new ArrayList<>(list);
         lists.add(path);
     }

    89.格雷编码

    题目；
    格雷编码是一个二进制数字系统，在该系统中，两个连续的数值仅有一个位数的差异。

给定一个代表编码总位数的非负整数 n，打印其格雷编码序列。格雷编码序列必须以 0 开头。

输入: 2
输出: [0,1,3,2]
解释:
00 - 0
01 - 1
11 - 3
10 - 2

思路：动态规划，从n-1的数的编码得到n的数的编码，其实就是在n-1的数的基础上在前面加上2的n-1次方，但是要逆序添加进list，

public List<Integer> grayCode(int n) {
        List<Integer> list=new ArrayList<>();
        list.add(0);
        for (int i = 0; i <n ; i++) {
            int add=1<<i;
            for (int j = list.size()-1; j >=0 ; j--) {
                list.add(list.get(j)+add);
            }
        }
        return list;
    }

125.验证回文串

题目：给定一个字符串，验证它是否是回文串，只考虑字母和数字字符，可以忽略字母的大小写。

说明：本题中，我们将空字符串定义为有效的回文串。

思路：先将字符串进行字符筛选，然后再判断是否是字符串

 public boolean isPalindrome(String s) {
        if(s==null||s.length()==0){
            return true;
    }
        s=s.toLowerCase();
        char[] ca=s.toCharArray();
        int index=0;
        for (int i = 0; i <ca.length ; i++) {
                if((ca[i]>='a'&&ca[i]<='z')||(ca[i]>='0'&&ca[i]<='9')){
                    ca[index++]=ca[i];
                }
        }
        int start=0;
        while (start<index-1){
            if(ca[start]!=ca[index-1]){
                return false;
            }
            start++;
            index--;
        }
        return true;
}
