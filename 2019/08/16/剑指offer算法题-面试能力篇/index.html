53.在排序数组中查找数字

题目一:统计一个数字在排序数组中出现的次数

思路：找到数字在数组第一次出现的位置，和第二次出现的位置，使用二分法求，当数字在出现的位置前面一个数不等于他，那么它就是第一个数，同理当数字在出现的位置后面一个数不等于他，那么它就是第一个数

  public int getNumberOfK(int[] arr,int k){
        if(arr==null||arr.length<=0){
            return -1;
        }
        int length=arr.length;
        int first=getFirstIndexOfK(arr,k,0,length);
        int last=getLastIndexOfK(arr,k,0,length);
        if(first==-1||last==-1)
            return -1;
        else
            return last-first+1;
    }
    int getFirstIndexOfK(int[] arr,int k,int start,int end){
        int mid=(start+end)/2;
        while (start<=end){
            if(arr[mid]>k){
                end=mid-1;
            }else if(arr[mid]<k){
                start=mid+1;
            }else {
                if(mid>0&&arr[mid-1]!=k){
                    return mid;
                }else {

                    end=mid-1;
                }
            }
            mid=(start+end)/2;
        }
        return -1;

    }
    int getLastIndexOfK(int[] arr,int k,int start,int end){
        int mid=(start+end)/2;
        while (start<=end){
            if(arr[mid]>k){
                end=mid-1;
            }else if(arr[mid]<k){
                start=mid+1;
            }else {
                if(mid>0&&arr[mid+1]!=k){
                    return mid;
                }else {
                    start=mid+1;
                }
            }
            mid=(start+end)/2;
        }
        return -1;

    }

55.二叉树的深度

题目一：求一棵二叉树的深度
思路：当输入根节点，树的深度就等于左子树的深度和右子树的深度中较大的加一，递归下去。

  int findTreeDepth(TreeNode pRoot) {
        if(pRoot==null){
            return 0;
        }
        int left=findTreeDepth(pRoot.left);
        int right=findTreeDepth(pRoot.right);
        return left>right?(left+1):(right+1);
    }

题目二：判断一棵树是不是平衡二叉树
平衡二叉树的定义：一个二叉树每个节点 的左右两个子树的高度差的绝对值不超过1。

思路：从叶节点开始往上遍历，判断每一层是否是平衡二叉树，这里要定义一个全局变量，isbalanced，若某一处不是平衡的，那么这棵树就不是平衡二叉树

private boolean isBalanced=true;
    public boolean isBalanced_Solution(TreeNode root){
        getDepth(root);
        return isBalanced;
    }
    public int getDepth(TreeNode root){
        if(root==null){
            return 0;
        }
        int left=getDepth(root.left);
        int right=getDepth(root.right);
        int depth=1+(left>right?left:right);
        if(left-right>=-1&&left-right<=1&&isBalanced){
            isBalanced=true;
        }else {
            isBalanced=false;
        }
        return depth;
    }

56.数组中数字出现的次数

题目一：数组中有两个数字恰好都只出现一次，其他的出现两次

思路：划分，将两个数字划分到两个不同的数组，然后异或，关键是怎么划分，划分的同时，两个相同的数字一定要划分到同一个数组。

  2. 遍历数组，如果这个数在indexOf1上是1，就划分到一个数组，如果是0，那就划分到另一个数组。

题目二：数组中有一个数字只出现一次，而其他数字出现三次

思路：将数组中所有数字某一位的bit位加起来对3求余，余数就是只出现一次数字某一位bit位的数字

public int singleNumber(int[] nums) {
      if(nums==null||nums.length<=0){
            return -1;
        }
        int[] bitSum=new int[32];
        int length=nums.length;
        for (int i = 0; i <length; i++) {
            int bitMask=1;
            for (int j = 0; j <32 ; j++) {
                if((nums[i]&bitMask)!=0){
                    bitSum[j]++;
                }
                bitMask=bitMask<<1;
            }

        }
        int result=0;
        for (int i = 31; i >=0 ; i--) {
            result=result<<1;
            result+=bitSum[i]%3;

        }
        return result;
    }

57.和为s的数字

题目一：输入一个递增排序的数组和一个数字S，在数组中查找两个数，使得他们的和正好是S，如果有多对数字的和等于S，输出两个数的乘积最小的。

思路：定义两个指针指向头尾，然后如果他们相加小于s那么头指针就++；大于s尾指针++,左右逼近

public ArrayList<Integer> FindNumbersWithSum(int [] array,int sum) {
                    ArrayList<Integer> result=new ArrayList<>();

        if (array==null||array.length<=1){
                return result;
            }
            int i=0;
            int j=array.length-1;
            int max=array[j]*array[j-1];
            while (i<j){
                if(array[i]+array[j]==sum){
                    if(array[i]*array[j]<max) {
                        result.add(array[i]);
                        result.add(array[j]);
                        max=array[i]*array[j];
                    }
                    i++;
                    j--;
                }else if(array[i]+array[j]<sum){
                    i++;
                }else {
                    j--;
                }
            }
            return result;
    }

其实这里并不需要判断他是不是最小的那个，当左右逼近时，乘积最小的那个数其实就是靠近最外围的。

题目二：和为s的连续整数序列
输入一个整数s，输出所有的和为s的连续整数序列

思路：定义两个指针small和big，数值等于1和2，然后从big++，比较此时的和与s，如果小于s，big++，如果大于s，就去掉small。让small指向2，循环往复，知道small大于或等于s的中间值。

public ArrayList<ArrayList<Integer> > FindContinuousSequence(int sum) {
         ArrayList<ArrayList<Integer> > result=new ArrayList<>();
            if(sum<3){
                return result;
            }
            ArrayList<Integer> sequence=new ArrayList<>();
            int small=1;
            int big=2;
            sequence.add(small);
            sequence.add(big);
            int middle=(sum+1)/2;
            int currentSum=3;
            while (small<middle){

                if(currentSum==sum){
                    ArrayList<Integer> path=new ArrayList<>(sequence);
                    result.add(path);
                    big++;
                    sequence.add(big);
                    currentSum+=big;
                }else if(currentSum<sum){
                    big++;

                    currentSum+=big;
                }else {

                    small++;

            }

    }

58.翻转字符串

题目一：翻转单词顺序，

输入： "i am a student"
输出： "student a am i"

这道题题目限制较松，不存在输入两边有空格的情况，中间的空格也不需要处理

 public String reverseWords(String s) {
        if (s == null) {
            return "";
        }
        s=s.trim();
        char[] str = s.toCharArray();
        int length = str.length;
        reverseString(str, 0, length - 1);
        int begin=0;
        for (int i = 1; i < length; i++) {
            if (str[i]==' '){
                if(i-begin<2){
                    continue;
                }
                reverseString(str,begin,i-1);
                begin=i+1;
            }
        }
        if(begin<length){
            reverseString(str,begin,length-1);
        }
        return String.valueOf(str);
    }

    private void reverseString(char[] str, int start, int end) {

        while (start < end) {
            char temp = str[start];
            str[start] = str[end];
            str[end] = temp;

            start++;
            end--;
        }
    }

            return "";
        }
        char[] chars=s.toCharArray();
        int length=chars.length;

    }

59.队列中的最大值

题目一：定一个数组和滑动窗口的大小，找出所有滑动窗口里数值的最大值。例如，如果输入数组{2,3,4,2,6,2,5,1}及滑动窗口的大小3，那么一共存在6个滑动窗口，他们的最大值分别为{4,4,6,6,6,5}； 针对数组{2,3,4,2,6,2,5,1}的滑动窗口有以下6个： {[2,3,4],2,6,2,5,1}，{2,[3,4,2],6,2,5,1}， {2,3,[4,2,6],2,5,1}， {2,3,4,[2,6,2],5,1}，{2,3,4,2,[6,2,5],1}， {2,3,4,2,6,[2,5,1]}。

思路：用一个双端队列，用于保存数字的下标

  1. 当队列为空时直接加入数值的下标，当当前的数值的下标减去队列里第一个元素的下标大于等于滑动窗口大小k时，将对头移出
  2. 移出比队列中当前元素还小的值
  3. 从第三个数开始将对头元素录入结果集。
    public int[] maxSlidingWindow(int[] num, int k) {
     if(num==null||num.length==0||k<=0){
         return new int[0];
     }
     ArrayDeque<Integer> queue=new ArrayDeque<>();
     int length=num.length;
     int[] maxValue=new int[length-k+1];
     int j=0;
     for (int i = 0; i < length; i++) {
         if (queue.isEmpty()){
             queue.add(i);
         }
         if(i-queue.peekFirst()>=k){
             queue.pollFirst();
         }
         while (!queue.isEmpty()&&num[queue.peekLast()]<=num[i]){
             queue.pollLast();
         }
         queue.add(i);
         if(i+2>k){
             maxValue[j++]=num[queue.peekFirst()];
         }


    }
    return maxValue;
}

题目二：请定义一个队列并实现函数max得到队列里的最大值，要求函数max、push_back和pop_front的时间复杂度都是O(1)。

思路：用两个双端队列，一个队列用于存放最大值，一个队列按先进先出的顺序push和pop，这里定义一个内部类，将数值和他的index联系起来，存放最大值的队列依据于滑动窗口。

class QueueWithMax {
ArrayDeque data=new ArrayDeque<>();
ArrayDeque maximum=new ArrayDeque<>();
int curIndex=0;
private class InternalData{
int number;
int index;
public InternalData(int number,int index){
this.number=number;
this.index=index;
}

}
public void push_back(int number){
    InternalData internalData=new InternalData(number,curIndex);
    data.add(internalData);
    while (!maximum.isEmpty()&&maximum.peekLast().number<=internalData.number){
        maximum.pollLast();
    }
    maximum.add(internalData);
    curIndex++;
}
public void pop_front(){
    if(data.isEmpty()){
        System.out.println("queue is empty");
        return;
    }
    InternalData curData=data.pollFirst();
    if(curData.index==maximum.peekFirst().index){
        maximum.pollFirst();
    }
}
public int max(){
    if(data.isEmpty()){
        System.out.println("queue is empty");
        return 0;
    }
    return maximum.peekFirst().number;
}

}

## 61.扑克牌中的顺子

public boolean isContinuous(int[] nums){
if(nums==null||nums.length==0){
return false;
}
Arrays.sort(nums);
int numberOfZero=0;
int numberOfGap=0;
int length=nums.length;
for (int i = 0; i < length&&nums[i]==0; i++) {
numberOfZero++;
}
int small=numberOfZero;
int big=small+1;
while (big<length){
if(nums[big]==nums[small]){
return false;
}
numberOfGap+=nums[big]-nums[small]-1;
small=big;
big=small+1;
}
return numberOfGap>numberOfZero?false:true;
}

## 65.不同加减乘除法做加法
题目：不同+-x÷实现加法
思路：用位运算，分三步走，第一步先异或，不考虑进位，第二步用与，计算进位，然后左移一步，第三再将异或后的值与进位相异或，那么这时候也有可能继续产生进位，循环下去。

public int add(int num1,int num2){
int sum;
int carry;
while (num2!=0){
sum=num1^num2;
carry=(num1&num2)<<1;
num1=sum;
num2=carry;
}
return num1;
}


## 66.构建乘积数组
题目:给定长度为 n 的整数数组 A，其中 n > 1，返回输出数组 output ，其中 output[i] 等于 nums 中除 nums[i] 之外其余各元素的乘积。

输入: [1,2,3,4]
输出: [24,12,8,6]

思路：这里不能用到除法，换个方向想，新建数组B，B[i]=A[0]+A[1]+A[2]+...+A[i-1]+A[i+1]+...+A[n-1],看成两部分，一一部分是A[0]+A[1]+A[2]+...+A[i-1],另一部分A[i+1]+...+A[n-1],那么分两部走，第一步从头开始A[0]+A[1]+A[2]+...+A[i-1]=B[i-1]*A[i-1],第二步从尾开始，不太好阐述，直接看代码

public int[] multiply(int[] A){
if(A==null||A.length==0) {
return new int[0];
}

   int length=A.length;
   int[] B=new int[length];
   B[0]=1;
   for (int i = 1; i <length ; i++) {
       B[i]=B[i-1]*A[i-1];
   }
   int temp=1;
   for (int i = length-2; i>=0 ; i--) {
       temp*=A[i+1];
       B[i]*=temp;

   }
   return B;

}
